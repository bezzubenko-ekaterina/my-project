# Путешествия

## Что здесь хранить

- Летние поездки: даты, маршруты, брони (без паролей и полных номеров документов в открытом виде)
- Визы: чек-листы документов, статус подачи
- Шаблоны списков вещей

## Что для меня важно

- **Сейчас:** много поездок запланировано на лето; **подаюсь на итальянскую визу** — нужны актуальные чек-листы и логика подготовки пакета (официальные требования всегда перепроверять на сайте консульства / VFS).
- **Стиль поездок:** *(бюджет / комфорт / актив — допиши)*
- **Типичная длительность:** *(по желанию)*
- **С кем езжу:** *(один / семья / друзья)*

**База:** живу в Москве, родом из Барнаула — при советах по логистике и «откуда лететь» это может иметь значение.

## Для AI

Помогай с маршрутами, списками, черновиками писем, структурой поездки. По визам: ориентируй на типичные пакеты и вопросы для самопроверки, с оговоркой перепроверить источники (сроки и списки документов меняются).
