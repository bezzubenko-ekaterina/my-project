# Моя жизнь — контекст для AI

Здесь лежит структура, чтобы ассистент понимал твои сферы жизни и не выдумывал детали.

## Профиль (актуально)

- **Где живу:** Москва (родом из Барнаула), часовой пояс MSK (UTC+3).
- **Главный фокус сейчас:** работа — бизнес-ассистент **у основателя** компании, международная логистика; открытие филиала в **Москве** (раньше ~10 лет — филиал в **Новороссийске**).
- **Параллельно важно:** чекапы, зал **через день**, контроль питания; учёба (диплом, выпуск); поездки летом и **подача на итальянскую визу**.

Подробности по сферам — в README соответствующих папок.

## Папки

| Папка | Назначение |
|--------|------------|
| `00_Inbox` | Быстрые заметки, идеи, ссылки — потом разбираешь по папкам |
| `01_Здоровье` | Тренировки, питание, врачи, анализы, сон |
| `02_Учёба` | Диплом, последний курс, выпуск, конспекты, дедлайны |
| `03_Путешествия` | Летние поездки, визы (в т.ч. Италия), маршруты, списки |
| `04_Финансы` | Бюджет, цели, подписки, крупные покупки |
| `05_Хобби` | Проекты, железо, прогресс, вдохновение |
| `06_Работа` | Бизнес-ассистент, логистика, запуск филиала в Москве |

## Ещё можно уточнить (по желанию)

- Сколько часов в неделю реально на спорт / диплом / подготовку к визе?
- **Как удобнее от AI:** короткие списки, длинный разбор, таблицы, готовый черновик?

Писать в чат: «открой `Моя_жизнь` и помоги с …» — контекст подтянется из файлов.
