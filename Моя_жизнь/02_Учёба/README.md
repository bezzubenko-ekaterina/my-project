# Учёба

## Что здесь хранить

- Текущие курсы и дедлайны
- Материалы по диплому: тема, план, главы, научрук, требования вуза
- Конспекты, шпаргалки, ссылки

## Что для меня важно

- **Статус:** последний курс, дописываю диплом, выпускаюсь.
- **Чему учусь / специальность:** *(допиши — тогда AI точнее поможет с формулировками и структурой)*
- **Дедлайны сдачи:** *(даты)*
- **Формат:** *(самоход / с научруком / очно-заочно)*
- **Язык материалов:** русский *(если иначе — укажи)*

## Для AI

Можно просить план глав, реформулировки, вопросы к тексту, сжатие длинных кусков, чек-лист перед сдачей. Не выдумывай факты из диплома — опирайся на то, что пользователь приложит или опишет здесь.
