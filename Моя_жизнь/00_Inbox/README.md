# Inbox

Сюда кидаешь всё необработанное: мысли, ссылки, скриншоты (описанием), задачи «потом».

## Как пользоваться

- Раз в неделю (или по настроению) просматривай и переноси в `01_Здоровье` … `05_Хобби` или удаляй.
- В README целевой папки дополняй факты — тогда AI не путается.

## Для AI

Если пользователь просит «разобрать инбокс», предложи разнести пункты по папкам и обновить соответствующие README.
