# Computer Use — AI кликает по экрану

## Что это

Computer Use — функция Claude Code, которая позволяет AI **видеть экран и кликать** по интерфейсу. AI буквально смотрит на скриншот, находит кнопки и элементы, и управляет мышкой/клавиатурой.

## Зачем это

Представь: ты написал веб-приложение через вайб-кодинг. Как проверить, что оно работает? Раньше — вручную кликать по кнопкам. Теперь:

```
Открой приложение в браузере.
Заполни форму регистрации тестовыми данными.
Нажми "Отправить".
Проверь что появилось сообщение "Успешно".
```

AI сам откроет браузер, заполнит поля, кликнет кнопку и проверит результат.

## Статус

⚠️ **Research Preview** — экспериментальная функция. Доступна:
- В Claude Code (терминальный агент)
- macOS only (пока)
- Требует разрешения на управление экраном

## Примеры использования

- **Тестирование UI** — AI проходит по сценарию: регистрация → логин → действие → проверка
- **Заполнение форм** — автозаполнение документов в браузере
- **Скриншоты** — AI делает скриншоты разных страниц для документации
- **Демонстрация** — AI показывает как использовать твоё приложение

## Ограничения

- Работает медленнее чем API-интеграции (скриншот + анализ + клик)
- Может ошибаться с мелкими элементами
- Не заменяет автотесты для продакшна
- Пока только macOS

## Когда использовать

✅ Быстро проверить UI после изменений
✅ Записать демо-сценарий
✅ Протестировать пользовательский сценарий

❌ Не для: production-тестирования, высоконагруженных сценариев
