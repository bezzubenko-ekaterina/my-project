# Полезные ресурсы по вайб-кодингу

## 📚 Гайды и документация

### Claude Code Handbook (FreeCodeCamp)
🔗 https://freecodecamp.org/news/claude-code-handbook/

Подробный гайд по Claude Code — от установки до продвинутых техник. На английском, но с примерами кода. Полезно для понимания как работают AI-агенты под капотом.

## 💬 Telegram-каналы

### @vibecoding_tg — Вайб-кодинг
Канал про вайб-кодинг на русском. Кейсы, инструменты, лайфхаки. Подписчики делятся опытом создания проектов через AI.

### @misha_davai_po_novoi — Миша, давай по новой
Канал о работе с AI-инструментами. Практические примеры, обзоры новых функций Cursor, Claude Code и других инструментов. Живой стиль, реальные кейсы.

## 🎯 Как использовать

1. Подпишись на каналы — следи за новостями и обновлениями
2. Handbook читай как справочник — когда нужен ответ на конкретный вопрос
3. Кейсы из каналов — пробуй повторить на своём проекте

## 🔄 Другие полезные ресурсы

- **cursor.com/learn** — официальная документация Cursor
- **docs.anthropic.com** — документация Claude API
- **lmarena.ai** — рейтинг AI-моделей (какая лучше для какой задачи)
