## Задание 3. Команды управления (сделай сам)

Добавь к боту команды — но НЕ копируй промпт из инструкции. Сформулируй сам:

1. Команда для показа статистики (сколько файлов в каждой папке)
2. Команда для показа последних сохранённых файлов
3. Ограничение: команды работают только для тебя (ADMIN_ID)

Подсказка: используй os.listdir() для подсчёта файлов.

**Готово, когда:** Команды показывают статистику, а чужой пользователь получает «Доступ запрещён».

## Задание 4. Бот под свою задачу (сделай сам)

Создай нового бота с нуля (новый токен от BotFather). Этот бот — под вашу задачу: для работы, проекта или личного использования.

Требования:
1. Приветствие /start с описанием что бот умеет
2. Подключи бесплатную AI-модель (Gemini, Grok или Hugging Face)
3. Напиши системный промпт — опиши своё дело, услуги, правила ответа
4. Свободный текст — бот отвечает через AI

Промпт для Cursor напиши сам. Опиши какой бот тебе нужен, какой функционал, для кого. Cursor всё сделает.

**Готово, когда:** Бот отвечает на /start, понимает вопросы и отвечает в контексте вашего дела.

## Задание 5. Запуск 24/7 (творческое)

1. Залей проект бота на GitHub (commit + push)
2. Размести бота на Hugging Face Spaces (бесплатно):
   - Создай Space на huggingface.co
   - Загрузи файлы
   - Добавь секреты в Settings → Variables
   - Проверь: бот отвечает без запущенного терминала

Если мощностей не хватает — арендуй VPS (от 200 руб/мес).

**Готово, когда:** Бот работает после закрытия терминала, код на GitHub.
