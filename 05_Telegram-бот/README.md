# Модуль 05. Умный Telegram-бот под вашу задачу

Создаём Telegram-бота с нуля — от регистрации в BotFather до запуска 24/7. Пять примеров ботов: автоответчик, AI-консультант, контекст-сборщик, модератор, автоматизатор.

## Чему научишься

- Регистрировать бота и получать токен
- Создавать проект бота в Cursor с правильной структурой
- Писать обработчики для текста, голосовых, файлов, фото
- Подключать бесплатные AI-модели (Gemini, Grok, Llama, Hugging Face)
- Запускать бота на Hugging Face Spaces (бесплатно 24/7)

## Файлы модуля

- **презентация.html** — слайды (открой в браузере)
- **инструкция.md** — пошаговый гайд (12 шагов)
- **практика.md** — 5 заданий от простого к сложному
- **материалы/** — готовый код, шаблоны, 10 готовых ботов

## Порядок прохождения

1. Посмотри презентацию
2. Пройди инструкцию
3. Выполни задания из практики

## Время

~3 часа на первое прохождение.

## Материалы

- `код_контекст_бота.py` — бот, собирающий контекст в папки
- `код_ai_бота.py` — бот с AI-консультантом (Gemini)
- `ТЗ_бота_шаблон.md` — шаблон ТЗ для создания бота через AI
- `вайбкодим_бота_с_нуля.md` — полный гайд от BotFather до запуска
- `готовые_боты/` — 10 готовых ботов (копируй и запускай)
