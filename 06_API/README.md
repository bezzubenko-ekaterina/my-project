# Модуль 06. API и интеграции

Подключаем внешние сервисы через API — Telegram, Google Sheets, Webhooks. Учимся собирать связки из нескольких сервисов через Cursor.

## Чему научишься

- Понимать что такое API, ключи, запросы
- Отправлять уведомления в Telegram через API
- Работать с Google Sheets через Cursor
- Использовать Webhooks для автоматических уведомлений
- Подключать MCP-серверы в Cursor
- Понимать разницу между RAG и контекстом

## Файлы модуля

- **01_презентация.html** — слайды (открой в браузере)
- **02_инструкция.md** — пошаговый гайд
- **03_практика.md** — 5 заданий от простого к сложному
- **04_дополнительные_материалы/** — готовые примеры, гайд по MCP

## Порядок прохождения

1. Посмотри презентацию
2. Пройди инструкцию
3. Выполни задания из практики

## Время

~2.5 часа на первое прохождение.
