# AI Sales Outreach — автоматизация продаж

**Источник:** https://github.com/kaymen99/sales-outreach-automation-langgraph (★251)

## Что это

Система AI-агентов для автоматизации полного цикла продаж:
1. **Поиск лидов** — собирает информацию о потенциальных клиентах
2. **Квалификация** — оценивает, подходит ли лид под ваш ICP (Ideal Customer Profile)
3. **Персонализация** — генерирует персональные письма/сообщения
4. **CRM** — записывает всё в HubSpot, Airtable или Google Sheets

## Как работает

Построено на LangGraph (фреймворк для AI-агентов). Цепочка агентов:
- **Research Agent** — ищет информацию о компании и контакте
- **Qualification Agent** — скоринг лида по критериям
- **Copywriter Agent** — пишет персонализированное письмо
- **CRM Agent** — сохраняет результат в вашу CRM

## Для кого

- Отделы продаж, которые делают холодные рассылки
- Фрилансеры, ищущие клиентов
- HR-агентства (адаптировать под поиск кандидатов)
- Любой бизнес с активными продажами

## Стек

Python, LangGraph, LangChain, OpenAI API
