# 200+ готовых n8n-автоматизаций

**Источник:** https://github.com/creativetimofficial/ct-n8n-templates

## Что это

Коллекция из 200+ готовых воркфлоу для n8n — платформы автоматизации без кода. Каждый шаблон — JSON-файл, который импортируется в n8n за 30 секунд.

## Категории шаблонов

- **AI и LLM** — интеграции с OpenAI, Gemini, Ollama, RAG-системы
- **Соцсети** — автопостинг в Instagram, Twitter, TikTok
- **Email** — автоматизация Gmail, рассылки, триггеры
- **CRM и базы** — Airtable, Notion, Google Sheets
- **Документы** — обработка PDF, генерация
- **Мессенджеры** — Telegram, WhatsApp, Slack, Discord
- **HR и рекрутинг** — автоматизация найма
- **Контент** — генерация коротких видео, клипов

## Как использовать

1. Установи n8n (бесплатно, self-hosted)
2. Скачай нужный JSON-шаблон
3. В n8n: Workflows → Import from File → выбери JSON
4. Настрой свои ключи API
5. Готово — автоматизация работает

## Для кого

Предприниматели, маркетологи, ассистенты — все, кто хочет автоматизировать рутину без программирования.
