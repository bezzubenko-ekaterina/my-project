# Notion AI CRM — AI-агенты для управления лидами

**Источник:** https://github.com/whassan99/Notion-AI-CRM-Agents

## Что это

Система AI-агентов, которая превращает Notion в умную CRM:
- **ICP Scoring** — оценивает лида по 100-балльной шкале
- **Market Research** — автоматический анализ рынка по каждому лиду
- **Signal Agent** — находит триггерные события (раунд инвестиций, найм, новый продукт)
- **Prioritization** — расставляет приоритеты (high/medium/low)
- **Action Agent** — рекомендует следующий шаг

## До и После

| До | После |
|----|-------|
| Ручной анализ каждого лида | Автоматический скоринг + исследование |
| Интуитивная квалификация | Оценка по критериям с обоснованием |
| Непонятно, кому писать первому | Приоритеты с объяснением |
| Пропущенные лиды | Автоматический флаг устаревших лидов |

## Для кого

- Бизнесы, которые ведут CRM в Notion
- Продажники и маркетологи
- HR-агентства (адаптировать под кандидатов вместо лидов)

## Стек

Python, CrewAI, Notion API
