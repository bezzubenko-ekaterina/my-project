# Frihet MCP — AI-управление бизнесом из Cursor/Claude

**Источник:** https://github.com/Frihet-io/frihet-mcp (35 инструментов)

## Что это

MCP-сервер, который подключает AI-ассистента (Claude, Cursor, Copilot) к бизнес-инструментам. 35 инструментов для управления бизнесом голосом/текстом:

```
Ты:     "Создай счёт для ТехноСтарт, 40 часов консалтинга по 5000₽/час, оплата до 1 марта"
Claude: Готово. Счёт INV-2026-089 создан. Итого: 200 000₽ + 20% НДС = 240 000₽.
```

## Возможности

- **Счета** — создание, отправка, отслеживание оплат
- **Расходы** — учёт и категоризация
- **Клиенты** — база клиентов, история взаимодействий
- **Продукты** — каталог услуг и товаров
- **Коммерческие предложения** — генерация КП
- **Налоги** — расчёт и соблюдение требований

## Как подключить

В Cursor добавить в `.cursor/mcp.json`:
```json
{
  "mcpServers": {
    "frihet": {
      "command": "npx",
      "args": ["-y", "@frihet/mcp-server"],
      "env": { "FRIHET_API_KEY": "ваш_ключ" }
    }
  }
}
```

## Для кого

Предприниматели, бухгалтеры, ассистенты — все, кто хочет управлять финансами и клиентами через AI.
