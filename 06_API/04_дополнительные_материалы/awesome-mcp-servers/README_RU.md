# Awesome MCP Servers — каталог всех MCP-серверов

**Источник:** https://github.com/appcypher/awesome-mcp-servers (★5200+)

## Что это

Самый полный каталог MCP-серверов — плагинов, которые подключают AI (Claude, Cursor, Copilot) к внешним сервисам. Сотни серверов по категориям:

## Категории

- **Файлы и документы** — работа с файлами, PDF, Google Drive
- **Базы данных** — PostgreSQL, MySQL, MongoDB, Supabase
- **Веб и браузер** — скрапинг, поиск, управление браузером
- **Код и DevOps** — GitHub, GitLab, Docker, CI/CD
- **Коммуникации** — Slack, Discord, Telegram, Email
- **Продуктивность** — Notion, Trello, Linear, Todoist
- **Финансы** — Stripe, банки, бухгалтерия
- **Маркетинг** — SEO, аналитика, соцсети
- **Данные** — Google Sheets, Airtable, Excel

## Зачем это в курсе

Когда ученик хочет подключить AI к конкретному сервису — ищет здесь готовый MCP-сервер вместо написания интеграции с нуля. Экономит часы/дни работы.

## Как использовать

1. Найди нужный MCP-сервер в README.md
2. Перейди по ссылке на GitHub
3. Следуй инструкции установки (обычно `npx` или `pip install`)
4. Добавь в конфиг Cursor/Claude
