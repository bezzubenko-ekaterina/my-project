# Модуль 04. Агенты и скиллы — AI-сотрудники для бизнеса

AI-сотрудник работает 24/7, стоит ~300₽ вместо 80 000₽ в месяц и учится с каждым запросом. В этом модуле создаём таких сотрудников.

## Чему научишься

- Находить задачи, которые стоит отдать AI (принцип «где задолбался — туда агента»)
- Создавать .mdc файлы — правила для AI-агентов, по 7-шаговой методологии
- Настраивать систему самообучения — агент улучшается с каждым использованием
- Выбирать модели под задачу: логика, креатив, код, объём
- Использовать Тройной прогон для важных задач
- Собирать библиотеку из нескольких агентов

## Файлы модуля

- **01_презентация.html** — слайды (открой в браузере)
- **02_инструкция.md** — пошаговый гайд (17 шагов, 6 частей)
- **03_практика.md** — 5 заданий от простого к сложному

## Материалы

- `готовые_mdc/` — 10 готовых .mdc файлов (скопируй в проект)
- `04_дополнительные_материалы/` — реальные кейсы и инструменты

## Порядок прохождения

1. Посмотри презентацию
2. Пройди инструкцию — шаг за шагом, не пропуская
3. Выполни все 5 заданий из практики

## Время

~2-3 часа на первое прохождение. После этого — создание каждого нового агента занимает 30-60 минут.
